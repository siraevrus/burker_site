# MiraShop | Burker - Интернет-магазин часов и украшений

Воссоздание фронтенда интернет-магазина MiraShop | Burker на Next.js с React и TypeScript.

## Технологический стек

- **Next.js 14** (App Router) - SSR и маршрутизация
- **React 18** - UI компоненты
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации
- **Zustand** - управление состоянием

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Структура проекта

```
burkerwatches/
├── app/                    # Страницы Next.js
│   ├── layout.tsx         # Главный layout
│   ├── page.tsx           # Главная страница
│   ├── cart/              # Страница корзины
│   ├── collections/       # Страницы коллекций
│   └── sale/              # Страница распродажи
├── components/            # React компоненты
│   ├── Header/            # Шапка сайта
│   ├── Footer/            # Подвал
│   ├── ProductCard/       # Карточка товара
│   ├── Filters/           # Фильтры товаров
│   └── Newsletter/        # Подписка на рассылку
├── lib/                   # Утилиты и логика
│   ├── types.ts           # TypeScript типы
│   ├── store.ts           # Zustand store
│   └── data.ts            # Данные товаров
└── public/                # Статические файлы
```

## Основные функции

- ✅ Главная страница с секциями Hero, Бестселлеры, Коллекции
- ✅ Каталог товаров с фильтрацией по коллекциям и цветам
- ✅ Корзина покупок с управлением состоянием
- ✅ Страницы коллекций (Macy, Olivia, Julia, Isabell, Ruby)
- ✅ Страница распродажи
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Анимации и плавные переходы
- ✅ Индикатор бесплатной доставки

## Команды

- `npm run dev` - запуск dev сервера
- `npm run build` - сборка для production
- `npm run start` - запуск production сервера
- `npm run lint` - проверка кода линтером
